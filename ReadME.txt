*-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-*
Название: MonoDrome.
Жанр: 1D Side-scrolling shooter.
Тип графики: 2D (DirectX9)
Версия: v1.0 - Release
Среда разработки: PureBASIC v4.30
Распространение: FreeWare OpenSource.
*-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-*

Управление:
Esc - Выход из игры.
R,К - Рестарт игры.
P,З,Space - Режим паузы.
TAB - Свернуть окно игры в tray.
1   - Выстрел циановым импульсом с первой станции.
2   - Выстрел желтым импульсом с первой станции.
3   - Выстрел красным импульсом с первой станции.
Q,Й - Выстрел циановым импульсом со второй станции.
W,Ц - Выстрел желтым импульсом со второй станции.
E,У - Выстрел красным импульсом со второй станции.
A,Ф - Выстрел циановым импульсом с третьей станции.
S,Ы - Выстрел желтым импульсом с третьей станции.
D,В - Выстрел красным импульсом с третьей станции.
Z,Я - Выстрел циановым импульсом с четвертой станции.
X,Ч - Выстрел желтым импульсом с четвертой станции.
C,С - Выстрел красным импульсом с четвертой станции.
